概念說明

【問題】投資決策者每天淹沒在聳動標題、重複推播與未驗證訊息中，浪費判斷力，錯失真正關鍵事件。 【解決方案】每日推播 ≤3 筆經 AI + 編輯雙層驗證的重大市場事件。首先，AI 模型整合政府經濟數據 API 與歷史股價影響度庫，以公開權重給每筆事件評分（≥70 分入選）；其次，編輯按分級 SLA（一級 50 分鐘、二級 120 分鐘）完成查證，署名、檢查清單與核查耗時全部透明；再次，與台灣事實查證中心合作，確保「重大事件」定義經過專業認證；最後，用戶投票達閾值時觸發事件重評，月度會議公開調整邏輯。 【用戶旅程】投資人早上 09:00、午間 14:00、晚間 20:00（小樣本驗證最優時段），收到推播通知。點開「驗證快報」頻道，每則快報附帶編輯署名、驗證標籤（可點擊展開資料來源、檢查清單、核查耗時），減少閱讀焦慮，增強決策信心。若對評級不同意，可投票意見；月度會議透明討論，強化參與感與信任度。 【價值主張】相比市場同業，本服務承諾：(1) 訊息信度可驗證；(2) 推播數量精準受控；(3) 編輯與流程透明；(4) 用戶意見直接影響邏輯調整。訂閱會員享專屬頻道，廣告主選擇高淨值用戶投放，溢價 20–30%（A/B 測試量化）。 【驗證機制】小樣本階段（100 筆事件），驗證 ≥70 分門檻的編輯同意度、最優推送時段、平均每日事件數。規模化後，邊際成本極低，主要投入為 AI 模型維護與編輯人力排班優化。
